### **주요 기능 설명**

#### 1. **전세계 무료 프록시 자동 수집**
- **지원 소스**: 10개 이상의 무료 프록시 웹사이트 및 API
- **자동 업데이트**: 정기적으로 프록시 목록을 새로고침하여 최신 상태 유지

#### 2. **국가별 ISP 자동 배정**
- 사용자가 국가를 선택하면 해당 국가의 주요 ISP 중 하나를 자동으로 랜덤 선택
- 한국: SKT, KT, LGU+
- 미국: AT&T, Comcast, Verizon, Spectrum
- 일본: NTT, SoftBank, KDDI
- 영국: BT, Virgin Media, Sky Broadband
- 베트남: VNPT, FPT, Viettel
- 태국: AIS, True, DTAC
- 필리핀: PLDT, Globe, Smart
- 독일: Deutsche Telekom, Vodafone, O2

#### 3. **서로 다른 대역대 할당**
- 각 ISP별로 정의된 IP 대역대 중에서 랜덤으로 선택
- 다양한 IP 대역대를 사용하여 자연스러운 트래픽 생성

#### 4. **성능 최적화**
- 수집된 프록시 중에서 응답 속도가 가장 빠른 프록시를 자동으로 선택
- 지속적인 성능 모니터링을 통해 불량 프록시는 자동 제거

#### 5. **UI 디자인**
- 직관적인 이모지 버튼을 사용하여 국가 선택 가능
- 실시간 로그 및 상태 표시
- 프록시 성능 분석 및 보고서 생성 기능

### **사용 방법**

1. **파일 저장**: 위 코드를 `GlobalFreeProxySystem.bas`로 저장
2. **BAS 실행**: Browser Automation Studio에서 파일 열기
3. **실행**: 원하는 국가 버튼을 클릭한 후 "시작" 버튼을 눌러 실행

이제 유료 프록시 서비스 없이도 전세계 무료 프록시를 활용하여 동일한 수준의 성능을 얻을 수 있습니다.
